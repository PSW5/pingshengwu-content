# pingshengwu-content

Public content repository for **pingshengwu.com** — the personal website of **Ping-Sheng Wu (吳秉聖)**, sound artist, computer music composer, and lecturer at the College of Arts, National Tsing Hua University, Taiwan.

This repository is the markdown source of truth for the public-facing website. All content here is also indexed by the live site at https://pingshengwu.com.

## Structure

- `works/` — individual artworks (sound installations, audiovisual works, performances)
- `writings/` — essays, artist statements, lectures, research notes
- `events/` — exhibition and performance history
- `bio.md`, `about.md`, `cv.md` — biographical material
